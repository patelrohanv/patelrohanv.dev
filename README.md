# patelrohanv.dev

## Local Dev Setup
### Install npm
[Install npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
### Install pnpm
```
npm install -g pnpm
```

### Install dependencies
```
pnpm install
```

### Run Dev Server
```
pnpm run serve
```
