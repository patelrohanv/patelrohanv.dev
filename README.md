# patelrohanv.dev

## Local Dev Setup
### Install npm
[Install npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
### Install yarn
```
npm install --global yarn
```

### Install dependencies
```
yarn install
```

### Run Dev Server
```
yarn run serve
```
